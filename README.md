# ✅ To-Do List Web App

A simple and interactive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.
This application helps users organize and manage their daily tasks efficiently by adding, completing, and removing tasks.

## ✨ Features

* ➕ Add new tasks easily
* ✔ Mark tasks as completed
* ❌ Delete tasks when finished
* 🎨 Clean and user friendly interface
* ⚡ Dynamic task updates using JavaScript
* 📱 Responsive design for better usability

## 🛠 Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript

## 🎯 Purpose of the Project

The purpose of this project is to practice **JavaScript DOM manipulation and event handling** while creating a simple productivity tool for managing daily tasks.
